# What's This
这是基于`React`、`React-Router`、`Redux`、`Koa2.0`实现的React服务端渲染方案。已实现首页、详情页。

# Getting Started
```
// 为了构建一个模板(dist/index.html)给koa模板引擎使用
npm run build:local

// 启动本地node服务器，这里设置的端口是4322
npm run server:dev

// 启动前端服务，这里的服务端口号是8088
npm start

// 访问 http://localhost:4322进行开发
```
